# 🧩 Reto HTML

## 📖 Descripción
Este repositorio contiene un reto práctico enfocado en el desarrollo web utilizando **HTML, CSS y JavaScript**.

El objetivo es poner en práctica los fundamentos del desarrollo frontend, incluyendo la estructura de una página, estilos visuales e interacción con el usuario.

## 🚀 Características
- 📄 Estructura de página con HTML
- 🎨 Estilos y diseño con CSS
- ⚡ Interactividad con JavaScript
- 🧠 Ejercicio práctico para reforzar conocimientos
- 💡 Código simple y fácil de entender

## 🛠️ Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📂 Estructura del Proyecto
```
reto_html/
│── index.html # Estructura principal
│── styles.css # Estilos del proyecto
│── script.js # Lógica e interacción
│── README.md # Documentación
```

## 📚 Objetivo del reto
Comprender la estructura básica de una página web
Aplicar estilos visuales con CSS
Manipular el DOM con JavaScript
Mejorar la lógica de programación en frontend


## ⚠️ Aviso

Este proyecto fue creado con fines únicamente educativos.
